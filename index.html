Data-Driven Pandemic Analytics for India

The COVID-19 pandemic has profoundly affected India's diverse states and populations in unique ways. This project synthesizes vast amounts of data using Tableau dashboards to track historical case trends, offer forecasting models, present vaccine distribution frameworks, and provide accessible interactive experiences.
